# ECF – Administrateur Système DevOps

## Objectif
Mise en œuvre d’un projet DevOps intégrant Terraform, Docker et Kubernetes.

## Technologies utilisées
- Debian 11
- Terraform
- AWS
- Docker
- Kubernetes (maquette)
- Git / GitHub

## Organisation du projet
Chaque dossier correspond à un composant du projet (backend, CI/CD, infrastructure, etc.).

## Sécurité
Les fichiers sensibles (tfstate, secrets) ne sont pas versionnés.
